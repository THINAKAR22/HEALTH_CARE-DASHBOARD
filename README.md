# 🏥 HealthWatch Dashboard

A modern health monitoring dashboard built using Flask and Python.

## 🚀 Features
- User authentication
- Health metrics tracking
- Dashboard with analytics
- Secure backend using Flask
- Virtual environment support

## 🛠 Tech Stack
- Python
- Flask
- HTML/CSS
- Bootstrap
- SQLite / MySQL (optional)
